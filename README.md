# LLM RAG .NET Project

## Description
Backend service using LLM with Retrieval-Augmented Generation (RAG).

## Tech Stack
- .NET 8
- OpenAI 
- PostgreSQL + pgvector
- EF Core

## Architecture
Clean Architecture + CQRS

## Features
- Semantic search
- Context-aware responses
- Document ingestion

## Getting Started
1. Clone repo
2. Configure appsettings.json
3. Run migrations
4. Execute API

## Endpoints
- POST /api/chat
- POST /api/documents
